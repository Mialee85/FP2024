## Inhalt html
Standardaufbau:
```
<!DOCTYPE html>
<html>

<head>
</head>

<body>
</body>

</html>
```

Grundlegende Blöcke:
- <p> - Blockelement mit Umbrauch (Paragraph)
- <div> - Blockelement ohne alles (Divisor / Unterteiler)
- <span> - Inline Element (Unterteilung innerhalb anderer Blöcke)
